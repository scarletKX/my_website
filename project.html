Twygator

A Web Application built during the summer of 2019. Once a Twitter user logged into Twygator through OAuth to Twitter, Twygator would get the most recent 5 statuses (tweets, retweets, etc) of every friend of the user, and display it with a 3x3 (i.e. 9 friends) per page layout.

Technology Used:
Front-End: HTML, CSS, JavaScript, Bootstrap 4
Back-End: Python, Flask framework, Jinja2, OAuth, SQLAlchemy, Twitter API

Source Code: twygator repository on GitHub
